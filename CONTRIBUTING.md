# Contributing

Thank you for considering contributing to this MuxCore module.

## Development

```bash
# Build
go build -o scheduler-cron ./cmd/module

# Test
go test ./...

# Lint
golangci-lint run
```

## PR Process

1. Branch from `main`
2. Keep changes focused — one concern per PR
3. All tests must pass
4. Update CHANGELOG.md with your changes

## Reporting Issues

Open an issue at https://github.com/Muxcore-Media/scheduler-cron/issues
